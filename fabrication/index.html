fabrication

Fabrication

A collection of things I have made, milled, wired, coded, folded, prototyped, and occasionally overcomplicated.

01 / PS70 final project

Automated Origami Folding Machine

My final project for PS70, a digital fabrication class. I built a mechanical-electronic system to automate part of the origami folding process, using servos, hinges, plywood boxes, holding mechanisms, soldering, laser cutting, and code.

02 / engineering design

ES 51 Final Project

My final project for ES 51, an engineering design class. My team built a competition robot that combined CAD, machining, drivetrain design, servo mechanisms, fabrication, testing, and mechanical analysis.

03 / machine building

Drawing Machine

A group-built ephemeral drawing machine that dispenses ink onto water, creating both plotted shapes from user input and secondary marbled paper patterns.

04 / cnc milling

CNC Milling

A fabrication project focused on computer-controlled cutting, toolpaths, material setup, and translating a digital design into a physical object.

05 / networking

Networking / IoT

A collaborative project using a microphone, buzzer, and LAN web server to respond to room noise with different notes and songs.

06 / kinetic sculpture

Kinetic Sculpture

A hands-on build exploring motion, mechanisms, and how simple physical systems can create repeated movement.

07 / mystery box

Wooden Mystery Box

A wooden box project exploring fabrication, assembly, tolerances, and the small design decisions that make a physical object actually work.
